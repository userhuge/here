# Look here for a guide
Helpful resources for decoding words of a BIP39 secret phrase, or alternatively letters of a passphrase.

# How to use:

Basically you can print the list which will guide you to set your backup offline, no electricity involved( after that printing).

For less exposed seed phrases or with a very clean machine, you can use the web<–todo-link>pag to guide you bit-by-bit with a nice highlighting and check-marking each finished bit with a mouse.
